ℹ️ About This Server

This project is a simplified multi-threaded web server written in Python.

It demonstrates key Operating System concepts including:

✅ Multi-threading for handling concurrent requests
✅ Thread synchronization for safe logging
✅ Memory management techniques in Python
✅ Basic scheduling of client connections
✅ File handling for serving static content

All requests and responses are logged, and performance can be tested using concurrent client scripts.
